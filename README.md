## WooCommerce NovaPay Gateway Plugin

A plug-and-play WordPress/WooCommerce payment gateway plugin that adds **NovaPay** as a checkout payment option on your WooCommerce store.  
Simply install, activate, enter your API keys, and you’re ready to start accepting payments.

---

### Installation
1. Download the plugin ZIP file.  
2. Log in to your WordPress Admin Dashboard.  
3. Navigate to **Plugins → Add New → Upload Plugin**.  
4. Upload the ZIP file and click **Install Now**.  
5. After installation, click **Activate**.

---

### Configuration
1. Go to **WooCommerce → Settings → Payments**.  
2. Find **NovaPay** in the list of payment gateways.  
3. Enter your required credentials and configuration values (API keys, mode settings, etc.).  
4. Save changes.

---

### RSA Keys Setup
You also need to update your RSA keys inside the plugin file:
**/woocommerce-novapay/includes/class-wc-gateway-novapay.php**


Replace the existing RSA keys with your own according to the NovaPay documentation.

---

### Features
- Simple plug-and-play installation  
- Integrated NovaPay payment gateway for WooCommerce  
- Easy configuration through the WooCommerce Payments tab  
- Secure RSA key support  
- Works with most WooCommerce-compatible themes
